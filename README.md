# 💜 HireMind AI

### Multi-Agent Recruitment Intelligence System

---

## 📌 Overview

HireMind AI is an AI-powered recruitment intelligence platform designed to automate candidate screening, semantic resume analysis, intelligent ranking, and recruiter insights using a multi-agent architecture.

The system leverages semantic embeddings, FAISS vector retrieval, AI agents, and MCP-style modular tool communication to help recruiters identify the most suitable candidates efficiently and accurately.

This project was developed as part of Kaggle’s **5-Day AI Agents: Intensive Vibe Coding Course with Google** capstone project.

# 🧠 Multi-Agent Architecture

The system uses multiple specialized AI agents:

| Agent                   | Responsibility                        |
| ----------------------- | ------------------------------------- |
| JD Analyzer Agent       | Processes job descriptions            |
| Resume Agent            | Extracts candidate information        |
| Ranking Agent           | Calculates semantic rankings          |
| Recruiter Insight Agent | Generates recruiter-friendly insights |
| Chat Assistant          | Answers recruiter queries             |

# 🔍 Workflow

1. Recruiter uploads a Job Description.
2. Multiple candidate resumes are uploaded.
3. Resume embeddings are generated using Sentence Transformers.
4. FAISS performs semantic similarity search.
5. Candidates are ranked based on semantic relevance.
6. Recruiter Insight Agent generates explainable insights.
7. Dashboard visualizes rankings and analytics.

---

# 💡 Why AI Agents?

Traditional ATS systems rely heavily on keyword matching and often fail to understand contextual relevance between resumes and job descriptions.

HireMind AI solves this by using:

* semantic reasoning,
* modular AI agents,
* vector search,
* explainable AI workflows.

This enables more intelligent and efficient hiring decisions.
